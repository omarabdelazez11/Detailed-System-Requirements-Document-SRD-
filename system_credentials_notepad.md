# AASTMT SYSTEM ACCESS NOTEPAD

This file contains the fixed usernames/IDs for the core roles in the AASTMT Room & Lecture Hall Management System.

## 1. System Administrator
- **Username:** admin_aastmt
- **Role:** Admin
- **Access:** Full system control, Settings, User Management, Global Analytics.

## 2. Branch Manager
- **Username:** manager_branch
- **Role:** BranchManager
- **Access:** Admin Dashboard, Master Calendar, Final Approval of Multi-Purpose Rooms.

## 3. Secretary
- **Username:** sec_office
- **Role:** Secretary
- **Access:** Personal Dashboard, Multi-Purpose Room Bookings, Calendar View.

## 4. Employee (Staff / Lecturer)
- **Username:** staff_member
- **Role:** Employee
- **Access:** Personal Dashboard, Lecture Room & Multi-Purpose Bookings, My Requests.

---
*Note: All passwords should follow the standard academic security protocol. (Default: Role@2026)*
